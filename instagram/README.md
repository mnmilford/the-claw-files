```

  the-dispatcher/live/  — (future) The Dispatcher's published images
```

All drafts, visual specs, captions, and iteration work live in the [deepfield-saga](/root/Projects/deepfield-saga/) repo.

## Workflow

1. Content is developed in `deepfield-saga/dispatches/{date}/{character}/{slug}/`
2. IG images are posted (build hype)
3. Approved finals are copied here as `{date}-{slug}-{n}.png`
4. Blog dispatch goes live on deepfieldtransmissions.com
